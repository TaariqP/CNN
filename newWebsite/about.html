What are Convolutional Neural Networks?

by TAARIQ PALA10th march 2018

The discipline of Artificial Intelligence is becoming an increasingly investigated area within mathematics and computer science as we attempt to create our own intelligence. Researchers believe that we can use the human brain as a model to replicate intelligence and effectively, emulate the brain. This involves the creation and use of neural networks.

Instead of emulating the entire brain which would be both time-consuming and computationally expensive (the human brain contains approximately 10^11 neurons), researchers have simulated a large array of neurons to create a network. This simulation is known as ‘deep learning.’ 

What are Neural Networks?

A neural network acquires knowledge through learning and this knowledge is stored within inter-neuron connection strengths known as synaptic weights. Neural networks are effectively weighted directed graphs in which the neurons act as nodes and the edges are connections between neuron inputs and outputs.

In standard neural networks, the input is received as a vector and transforms it through a series of layers that are hidden. This input are notated by x(n) where n is the number of inputs and each of these layers are made up of a set of neurons. Each input is then multiplied by its corresponding weight. The inputs x and w can be positive or negative. As in Figure 2, the perceptron consists of weights, an activation function and a summation processor. All of the inputs are summed and passed to the activation function. An activation function is a mathematical function mapping a set of inputs to a set of outputs. The simplest example of an activation function would be a “step function” which outputs a 1 if the value is above a certain threshold, 0 otherwise. Other mathematical functions can be used.


The idea behind the training of a neural network is simple. We provide the neural network with training data, and the neural network will learn from that data and apply this to future data. For example, if we show the network an image of an dog we can expect the network to be able to identify correctly whether the next image is that of a dog as opposed to something else.

What are Convolutional Neural Networks?


Convolutional Neural Networks (CNN), like standard neural networks, are made up of neurons with learnable weights and biases (threshold processors). They are effectively a particular class of feed-forward neural networks. Unlike standard neural networks as described above, the input for a CNN is shown below.


This is known as an input layer or an input volume and it consists of a matrix of pixel values. In the example in Figure 3, the width and height are 32 and the depth is 3. The depth represents RGB channels. The first layer in a CNN is always a convolutional layer. The objective of the convolutional layer is to extract features of the input volume.


A convolutional layer has a set of matrices. These matrices are called filters or kernels. Filters are used to detect low level features such as curves and edges. A good way to understand this is to visualise a flashlight on the volume covering a 5 x 5 area. It has the same depth as the volume also. This flashlight is the filter and it slides across all the areas of the input image. The region covered by the flashlight is called the receptive field.  The filter is an array of weights. It traverses or convolves around the volume, multiplying values in the filter with values in the volume. These multiplications are summed up to produce a single number. We repeat this for the entire volume. The remaining numbers is known as an activation map. This is outputted by the convolutional layer. Typically the convolutional neural networks consists of many convolutional layers. The output of one layer is the input of another.


As you might expect, we require higher level features to be detected in order to accurately identify the object an image represents. Thus the first convolutional layer will have detected low level features and its activation map will be passed to the second layer. The second layer will detect higher levelled features.


Having detected some of the high level features, the final stage is to have a full connected layer at the end of the network. It will take an input volume and produce an X-dimensional vector, where X is the number of classes that the computer can choose from. For example, if you were to determine what digit is stored within an image you would have 10 possible classes. Each element of the vector represents the probability of a specific class. Thus the fully connected layer takes the final activation map (having been through all the filters) and determines which features best correlate to a particular class. For example, if the image represented a dog the activation map will have high values where it represents four legs, paws and a tail.


So how do we train the CNN to differentiate a particular object from others? How does it learn?The computer is able to adjust its filter values through a training process known as back propogation. This will be discussed in further detail in the Applications section of this website.
